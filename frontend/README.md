# NeuroFlow (frontend)

Vite + React + Tailwind dashboard for **NeuroFlow: Intelligent Traffic Control System using Neuro-Fuzzy Logic**.

Design source: [Figma — NeuroFlow](https://www.figma.com/design/nqRKeZ8jf7gbK9ujbBS1dM/NeuroFlow).

## Commands

```bash
npm install
npm run dev      # http://localhost:5173
npm run build
npm run preview
```

See the repository root `README.md` for full stack setup.
